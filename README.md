turbo-dangerzone
================

Simple Example of a Maven Project.

Instructions
===

 - [Install and Run Maven]

```
git clone https://github.com/gitjonathan/turbo-dangerzone.git
cd turbo-dangerzone
mvn clean install
cd target
java -jar turbodangerzone-0.0.1-SNAPSHOT.jar
```

You should see
```
Hello World!
```
